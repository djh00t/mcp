🔗 ChromaDB MCP Server

HTTP/SSE Model Context Protocol Interface

🚀 About this Server

This is a production-ready HTTP/SSE wrapper for the ChromaDB Model Context Protocol server. It enables web applications and HTTP clients to interact with ChromaDB through the standardized MCP interface.

Framework: Custom Express.js wrapper with Docker isolation
Transport: HTTP + Server-Sent Events (SSE)
Backend: ChromaDB vector database
Protocol: Model Context Protocol (MCP) 2024-11-05
